# TASK-118: Git workflow tools (git_status, git_log, git_diff)

**Priority**: high
**Status**: ready
**Category**: feature
**Source**: QA Audit Wave 4 — Feature Gap Analysis

## Description
World Tree lacks built-in git tools. Users must switch to terminal for basic git operations. Add native git status, log, and diff display integrated into the Command Center or project context.

## Acceptance Criteria
- [ ] Git status viewable per project (staged, unstaged, untracked)
- [ ] Git log browsable with commit messages and authors
- [ ] Git diff viewable with syntax-highlighted inline diff
- [ ] Integrates with existing project cards in Command Center
