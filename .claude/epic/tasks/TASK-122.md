# TASK-122: Auto-decision logging in memory system

**Priority**: medium
**Status**: ready
**Category**: feature
**Source**: QA Audit Wave 4 — Feature Gap Analysis

## Description
Architectural decisions made during conversations should be automatically detected and logged to the memory system. Currently decisions are only captured if manually logged.

## Acceptance Criteria
- [ ] Detect decision patterns in conversation messages
- [ ] Auto-log decisions with context and rationale
- [ ] Decisions appear in Compass knowledge base
- [ ] User can review and approve/reject auto-logged decisions
