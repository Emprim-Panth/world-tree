# TASK-046: Fix auto-scroll: conversation view doesn't scroll to show new messages

## Description

When new messages arrive (streaming or external), the view does not auto-scroll to keep the latest content visible. Root cause: onChange(of: sections.count) uses the unreliable isAtBottom @State (tracked via onAppear/onDisappear) instead of viewModel.isScrolledToBottom from ScrollBottomTracker. Also, onChange(of: isProcessing) only scrolls when processing starts, not when it completes.
