## Description

FRD-005 | Vertical scroll list of messages. User/assistant visually distinct (background/alignment). System messages as inline dividers. Markdown: bold, italic, code inline, code blocks with syntax highlighting + copy + horizontal scroll, lists, headings, links. Paginate 50/page, scroll-to-top loads more.

## Implemented
- StreamingSectionView: replaced plain Text() with MarkdownCodeFenceView — streaming content now renders with markdown (bold, italic, code blocks, links)
- System messages: rendered as inline dividers (HStack with flanking Dividers) instead of full message bubbles
- MarkdownCodeFenceView prose rendering: switched from .inlineOnlyPreservingWhitespace to .full — enables headings (#, ##), bullet/numbered lists
- Code blocks: already fully handled via CodeBlockView → ArtifactRendererView (highlight.js, copy button, horizontal scroll)
- Fallback: AttributedString parse failures fall through to plain Text on every prose segment
- User messages stay plain text (no markdown parsing)
