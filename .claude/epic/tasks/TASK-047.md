## Description

Two UI bugs reported:
1. Opening an old conversation starts at the top instead of the bottom. Root cause: ScrollBottomTracker sets isScrolledToBottom=false immediately because scroll is at top with content below, blocking onChange(of: sections.count) auto-scroll.
2. Sidebar sort/filter Menu button is invisible. Root cause: Menu + .menuIndicator(.hidden) on macOS 15 (Darwin 25) loses its hit area/visibility. Fix: add .menuStyle(.button).buttonStyle(.plain).
